## Introduction

This project is a submission for HackJam 2025 at USF. We followed the UI/UX track and built a frustrating UI experience inspired by The Impossible Game and other bad UI designs.


## Features

- **Log in Troll:** Log in page with deceiving sign up options and pranks embedded into it.

- **Sign Up Page:** Functional sign up page with live validation and requirements for all input fields. 

- **Click The Button Page:** Near Impossible find the button game where the user needs to find and click the button four times before advancing.
  
- **Slider Page:** Confusing slider page that only advances if the user holds ball exactly in the middle.
